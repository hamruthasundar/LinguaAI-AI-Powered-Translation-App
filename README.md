# LinguaAI – AI-Powered Translation App

LinguaAI is a multilingual mobile translation application built using React Native, Expo, TypeScript, AsyncStorage, and Google's Gemini API.

## Features

* Real-time AI-powered translations
* Support for multiple languages
* Translation history
* Modern mobile UI
* Language switching
* Offline history storage

## Technologies Used

* React Native
* Expo
* TypeScript
* AsyncStorage
* Gemini API

## Installation

```bash
npm install
npx expo start
```
